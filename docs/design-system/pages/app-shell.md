# App Shell

## Page or Surface Role

The app shell provides stable framing for the whole product.

It should help users feel oriented across routes without competing with the dominant job of the current page.

## Dominant Job

Orient the user, provide route structure, and frame the active page clearly.

## Primary Priorities

- Stable navigation
- Clean page framing
- Strong title-region rhythm
- Predictable spacing
- Coherent light and dark theming

## Required Qualities

The shell should feel:
- Calm
- Lightweight
- Consistent
- Helpful
- Visually subordinate to the page content

## Allowed Emphasis

Allowed:
- Clear active-route state
- Consistent page containment
- Subtle identity and personality cues

Avoid:
- Loud shell chrome
- Oversized navigation dominance
- Excessive decorative surfaces
- Shell elements that visually outrank page content

## Layout Guidance

- Keep the shell structurally consistent across routes
- Ensure page padding feels generous but efficient
- Keep navigation scannable and clear
- Let the content area remain the main focus

## Surface Guidance

The shell may use surface treatment, but it should not feel like a stack of boxed modules before the user even reaches the page.

Reduce shell surface weight when:
- The border is decorative rather than structural
- The shell competes with page content
- Multiple shell elements create redundant boxing

## Mobile Behavior

On mobile:
- Preserve route clarity
- Keep primary content quickly reachable
- Avoid a shell that consumes too much vertical space
- Ensure navigation remains discoverable and usable

## Trust Notes

The shell should suggest product cohesion, not false completeness.

It should frame the current product honestly without making unfinished areas feel more powered than they are.

## Review Questions

1. Does the shell help orientation without stealing attention?
2. Is the active route clear?
3. Does page content still feel dominant?
4. Does the shell still feel like Food Run without becoming decorative noise?
5. Is any shell surface heavier than it needs to be?
