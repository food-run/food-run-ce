# Food Run Trademark Policy

This policy describes how you may use the Food Run name, logos, and related branding ("Food Run Marks").

Food Run Community Edition (CE) is open source software, but trademark rights are separate from copyright and licensing.

## 1) Who owns the trademarks

The Food Run Marks are owned by the Food Run project owner ("Project Owner"). This includes:

- the name "Food Run"
- logos, icons, and brand assets associated with Food Run
- any confusingly similar names or logos used to imply official status

## 2) Permitted uses (no written permission needed)

You may use the Food Run Marks to:

- refer to the project in a factual way (e.g., "compatible with Food Run")
- link to the official repository
- describe unmodified builds of Food Run CE
- write tutorials, reviews, or articles about Food Run
- mention Food Run in talks and educational materials

## 3) Uses that require written permission

You must obtain written permission from the Project Owner to:

- use the Food Run Marks in a product name, company name, or domain name
- use the Food Run Marks in marketing that suggests affiliation, endorsement, or official status
- use the Food Run Marks on merchandise (shirts, stickers, etc.)
- use the Food Run Marks in a way that could confuse users about the source of software or services

## 4) Forks and modified versions

If you create a fork or modified version:

- you must use a different name and branding
- you must not use Food Run logos or confusingly similar branding
- you must clearly state that your project is a fork and is not the official Food Run project

Examples of acceptable naming:

- "MyMealPlanner (a fork of Food Run CE)"

Examples of unacceptable naming:

- "Food Run Plus"
- "Food Run Enterprise"
- "Food Run Official"

## 5) Hosting and services

If you host a service based on Food Run CE:

- you must not brand it as "Food Run" without permission
- you must not imply endorsement or partnership
- you must clearly identify yourself as the service provider

## 6) Enforcement

The Project Owner may revoke permission to use the Food Run Marks at any time if:

- the use is misleading
- the use harms users or the project reputation
- the use violates this policy

## 7) Contact

For trademark permission requests, see the contact method listed in `GOVERNANCE.md`.
