# Async Splitting

## TL;DR

This skill helps divide work into parallel lanes with minimal file overlap and clearer sequencing. It improves speed without increasing merge conflict risk.

## What I Do

- Separate read-heavy from write-heavy work
- Minimize file overlap
- Define sequencing and handoff points
- Reduce coordination debt in parallel execution

## When to Use Me

Use this skill when the team wants to parallelize a deliverable or a large task safely.
