# Coordination Handoff

## TL;DR

This skill keeps live coordination readable and current. It helps structure task notes, checkpoint notes, and handoffs so work can continue cleanly across agents.

## What I Do

- Structure task notes
- Structure handoff notes
- Structure checkpoint notes
- Preserve thread continuity between agents and humans

## When to Use Me

Use this skill whenever a work unit changes state or ownership.
