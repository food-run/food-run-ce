# Review Rubric

## TL;DR

This skill runs the repo’s review standard consistently. It helps catch plausible-but-dangerous output before it becomes mergeable.

## What I Do

- Run the repo review questions consistently
- Catch drift, duplication, and boundary collapse
- Return approve or reject with concrete reasons
- Highlight protected-path and ⚠️ Hotspot-File concerns

## When to Use Me

Use this skill during review, verification, or before asking for a checkpoint commit.
