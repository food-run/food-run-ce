# Planning Reader

## TL;DR

This skill helps load the smallest planning set needed for the current work. It keeps sessions focused and prevents unnecessary packet overload.

## What I Do

- Identify the smallest planning set needed for the current task
- Summarize constraints and acceptance criteria
- Prevent overloading the session with irrelevant planning text
- Flag planning conflicts early

## When to Use Me

Use this skill at the start of any sprint, deliverable, or task.
